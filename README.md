# 🏏 IPL Capstone Project

## 📌 Project Overview

This project analyzes IPL (Indian Premier League) data to extract meaningful insights about teams, players, and match performance.

## 🎯 Objectives

* Analyze team performance over seasons
* Identify top players (batsmen & bowlers)
* Explore match-winning factors
* Visualize IPL trends using graphs

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

## 📂 Project Files

* `IPL_Capstone_Project.ipynb` → Main analysis notebook
* Dataset files (CSV)

## 📊 Key Insights

* Most successful IPL teams
* Top run scorers and wicket takers
* Toss impact on match results
* Venue-based performance analysis

## ▶️ How to Run

1. Download the notebook
2. Open in Jupyter Notebook
3. Run all cells
